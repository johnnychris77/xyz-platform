🚀 What This Project Delivers

  • CI/CD pipeline with Jenkins
  • Docker container builds for microservices
  • Helm-based Kubernetes deployment
  • Terraform-managed AWS EKS and ECR resources
  • GitHub Actions for IaC automation

🔧 Jenkins Pipeline Stages

1. Checkout from GitHub
2. Build using Maven
3. Unit testing and reporting
4. Docker image creation and push
5. Kubernetes deployment (Test → Preprod → Prod)
6. Email/Slack notifications

provider "aws" {
  region = "us-east-1"

}

🧪 GitHub Actions Workflow

name: Terraform Apply
on:
  push:
    paths:
      - 'terraform/**'
jobs:
  terraform:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - run: terraform init
      - run: terraform apply -auto-approve

📚 How to Use

Clone the repo, configure Jenkins with your GitHub project, deploy to Kubernetes using Helm, and provision infrastructure with Terraform.
